# 批量插入

MyBatis 批量插入常用三种方式。

## 方式对比

| 方式 | 原理 | 适用 |
|------|------|------|
| **foreach 拼接** | 一条 SQL 多组 values | 数据量中等，注意 SQL 长度限制 |
| **ExecutorType.BATCH** | JDBC 批量提交 | 大量数据，减少网络往返 |
| **第三方批量** | 如 MySQL `rewriteBatchedStatements` | 配合 BATCH 模式 |

## 1. foreach（多值插入）

```xml
<insert id="batchInsert">
  INSERT INTO user (name, age) VALUES
  <foreach collection="list" item="u" separator=",">
    (#{u.name}, #{u.age})
  </foreach>
</insert>
```

## 2. BATCH 执行器

```java
try (SqlSession session = sqlSessionFactory.openSession(ExecutorType.BATCH)) {
    UserMapper mapper = session.getMapper(UserMapper.class);
    for (User u : list) {
        mapper.insert(u);
    }
    session.commit();
}
```

## 注意

- foreach 单条 SQL 过大可能超过 `max_allowed_packet`（MySQL）。
- 超大批量建议分批 + BATCH，或 LOAD DATA 等专业方案。
- PageHelper 是**分页**插件，不负责批量插入。

## 面试要点

- foreach 一条 SQL 多行；BATCH 多条语句批量提交。
- 大批量注意事务、批次大小和数据库包大小限制。

## 面试一句话

> 批量插入可用 foreach 拼多值 SQL，或用 ExecutorType.BATCH 走 JDBC 批处理，超大量要分批提交。
