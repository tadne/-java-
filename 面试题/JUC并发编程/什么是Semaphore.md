# 什么是 Semaphore

计数信号量，控制同时访问资源的**线程数**（或许可数），基于 **AQS 共享模式**。

## 常用 API

| 方法 | 作用 |
|------|------|
| `acquire()` | 获取 1 许可，不足则阻塞 |
| `acquire(n)` | 获取 n 个 |
| `release()` | 释放许可 |

## 与 synchronized

| | Semaphore | synchronized |
|---|-----------|--------------|
| 同时持有者 | 多个（≤许可数） | 一个 |
| 用途 | 限流、连接池 | 互斥 |

## 面试一句话

> Semaphore 是共享 AQS，state 表示许可数，用于限流与资源池，非互斥锁。
