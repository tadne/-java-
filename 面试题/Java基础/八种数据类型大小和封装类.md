# 八种数据类型大小和封装类

## 基本类型

| 类型 | 字节 | 位 | 包装类 |
|------|------|-----|--------|
| byte | 1 | 8 | Byte |
| short | 2 | 16 | Short |
| int | 4 | 32 | Integer |
| long | 8 | 64 | Long |
| float | 4 | 32 | Float |
| double | 8 | 64 | Double |
| char | 2 | 16 | Character |
| boolean | 未严格规定 | — | Boolean |

## 缓存（Integer 等）

- `Integer.valueOf(-128~127)` 使用缓存池（可配置上限）

## 面试一句话

> 8 种基本类型各有固定大小与对应包装类；小整数 Integer 有缓存。
