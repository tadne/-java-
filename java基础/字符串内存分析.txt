字符串

	栈：	方法运行进栈，执行完毕出栈
	堆：	new的对象都存在其中
	方法区:	加载字节码文件，临时存储

	串池StringTable：	new创建字符串对象的时候就将对象放到堆中，串池是在堆中的一个空间		
					
				String name=”abc“;
					这样就会先检查串池中是否有”abc“这个字符串，
						如果有就直接将其引用赋值给name，
						如果没有就在串池中创建一个新的字符串，并将其引用赋值给name。
				String name=new String("abc");
						它会在堆中和串池创建字符串对象,只有引用它时
						才会检查它是否在串池中存在并创建对象
					也就是说  :	String name="abc";和String name=new String("abc")获取的对象来源不同
								前者是串池中的,后者是堆中的


		也就是说,只要创建字符串对象,那一定会在串池中创建,而是否在堆中创建要看有没有使用new关键字

		只要使用new关键字,那么实际引用的就是堆中字符串

		没有使用new关键字,那么实际引用的就是串池中字符串
