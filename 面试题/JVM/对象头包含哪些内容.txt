Java 对象头（Object Header）
	用于存储对象的元数据信息的数据结构，位于对象的内存空间的最前端，占用一定的字节大小。

Java 对象头的内容主要有以下几个部分：

	Mark Word：	用于存储对象的运行时状态信息的字段，
			占用 32 位或 64 位，取决于操作系统和 JVM 的设置。

		存储的信息包括:		对象的哈希码、GC 分代年龄、锁状态标志、
					线程持有的锁、偏向线程 ID、偏向时间戳等。

		其中存储的信息会随着对象的状态变化而变化，
		如在 GC 时会存储 GC 标记信息，在同步锁时会存储锁信息等。

	Class Metadata Address：
		Class Metadata Address 用于存储对象所属类的元数据信息地址的字段，
		它占用 32 位或 64 位，取决于操作系统和 JVM 的设置。
	
		Class Metadata Address 中存储的地址指向方法区中的类元数据信息，
			包括类名、访问修饰符、字段描述符、方法描述符等。

		Class Metadata Address 中存储的地址是固定不变的，可通过它来确定对象的类型。


	Array Length：
		Array Length 用于存储数组长度的字段，
		只有在对象是数组类型时才存在，占用 32 位。
		Array Length 中存储的值是数组的长度，也就是数组中元素的个数。
		Array Length 中存储的值是固定不变的，可以通过它来确定数组的大小。
