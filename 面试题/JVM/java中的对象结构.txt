Java 的对象结构是指 Java 对象在内存中的组成和布局，


主要包括以下三个部分：

	对象头（Object Header）：
		对象头存储了对象的运行时数据，
		如对象的哈希码、GC 分代年龄、锁状态、线程持有的锁、偏向线程 ID 等信息。
		对象头的大小和结构可能因为 JVM 的不同而有所差异，一般占用 32 或 64 位2。

	实例数据（Instance Data）：
		实例数据存储了对象的实例字段，也就是对象的属性值。
		实例数据的大小取决于对象的类定义和字段类型，不同类型的字段占用不同的字节数2。

	对齐填充（Padding）：
		对齐填充是为了满足 JVM 对对象内存地址对齐的要求，使得对象的大小是 8 字节的整数倍。
		对齐填充不存储任何有效信息，只是一些占位符2。
