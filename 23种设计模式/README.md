## 常见混淆对比

| 对比 | 区别 |
|------|------|
| 装饰 vs 代理 | 装饰侧重**功能叠加**；代理侧重**访问控制** |
| 策略 vs 状态 | 策略由客户端**主动切换**；状态由对象**内部自动切换** |
| 桥接 vs 适配器 | 桥接**设计阶段**解耦；适配器**事后**兼容旧接口 |
| 组合 vs 装饰 | 组合是**树形整体-部分**；装饰是**一层层包装** |
| 工厂方法 vs 抽象工厂 | 工厂方法创建**一种**产品；抽象工厂创建**一族**产品 |

## 面试一句话

> 23 种模式 = 5 创建 + 7 结构 + 11 行为；先记 SOLID，再记单例/工厂/代理/装饰/观察者/策略在 Spring 与 JDK 中的落地。
