Java有什么特点？
	面向对象：	万物皆对象，将对象的状态以属性表示，对象的行为用方法表示

	平台无关性:	引入jvm虚拟机,java的语言被编译为与平台无关的class文件，由虚拟机解释执行
			实现一次编写，到处运行。

	高性能：		先编译后解释的语言，虽然不如全编译语言快。
			但是java支持即时编译技术，在运行时可以将字节码转换为cpu机器码，实现全编译

	安全：		Java可编写防病毒和可修复的系统，通常被用在网络环境中，
			Java提供了一个安全机制以防恶意代码的攻击，从而可以提高系统的安全性。

	健壮：		强类型机制、异常处理、垃圾的自动收集等是Java程序健壮性的重要保证

	简单：		提供了丰富类库和API文档，及第三方开发包工具包，有大量的基于Java的开源项目

	解释执行：	Java 程序运行时会被编译成字节码文件，然后可在Java环境的操作系统上运行。
			运行时，Java解释器对字节码解释执行，执行中加入的类在连接时被载入到运行环境

	多线程：		Java提供多线程间的同步机制，这些同步机制可以保证不同线程间能正确的共享数据
		
	分布式		Java 支持 Internet开发，Java基本应用编程接口中就有一个网络应用编程接口，
			提供了网络应用编程的类库，包括 URL、URLConnection、Socket 等。
