TCP首部:			包括固定20个字节以及可变长的部分
		{
			16位源端口号    16位目的端口号
			32位序号
			32位确认号
			4位头部长度,6位保留,16位窗口大小
			16位校验和,16位紧急指针
		}(以上20字节)	
			选项,最多40字节
			
UDP首部			只有8个字节,由4个字段组成,每个在字段长度都是2个字节
				UDP报头结构:		携带数据大小长度 - 8
		{
			16位源端口号    	16位目的端口号
			16位UDP长度		16位UDP校验和
		}(以上8字节)
			数据	
		



HTTP请求格式:	
		请求行,请求头,空行,请求数据四个部分

		请求报文一般格式:
	请求行:		请求方法  空格  URL  空格  协议版本  回车符  换行符   
	请求头:		头部字段名:	值	回车符  换行符   
				头部字段名:	值	回车符  换行符   
				...
				头部字段名:	值	回车符  换行符   
				回车符  换行符
	请求数据:	



HTTP响应格式:
		状态行,消息报头,空行,响应正文

		一般格式:
	状态行:		版本  空格  状态码  空格  短语  回车符  换行符 
	响应报头:		头部字段名:	值	回车符  换行符   
				头部字段名:	值	回车符  换行符   
				...
				头部字段名:	值	回车符  换行符   
				回车符  换行符	
	响应正文
