# 重载和重写的区别

## 对比表

| 维度 | 重载（Overload） | 重写（Override） |
|------|------------------|------------------|
| 位置 | 同类中 | 子类覆盖父类方法 |
| 方法名 | 相同 | 相同 |
| 参数列表 | **必须不同** | **必须相同** |
| 返回类型 | 可不同 | 协变返回（JDK 5+） |
| 访问修饰符 | 无限制 | 不能更严格 |
| 异常 | 无限制 | 不能抛出更广受检异常 |
| 绑定 | 编译期（静态） | 运行期（动态） |
| `static`/`final` | 可重载 | 不能重写 |

## 记忆

- **重载**：一个类里“同名不同参”，编译器根据实参选方法
- **重写**：子类提供父类方法的新实现，多态的基础

## 面试一句话

> 重载是同类同名不同参、编译期绑定；重写是子类覆盖父类、运行期多态。
