# Java 有什么特点

## 核心特点

| 特点 | 说明 |
|------|------|
| 面向对象 | 封装、继承、多态、抽象 |
| 跨平台 | 源码 → 字节码 → JVM 在各 OS 解释/ JIT |
| 自动内存管理 | GC 回收堆对象 |
| 强类型 | 编译期类型检查 |
| 异常机制 | 受检与非受检异常体系 |
| 多线程 | 语言级线程与 JUC 工具 |
| 丰富生态 | 标准库、Maven、Spring 等 |

## 执行模型

- **编译 + 解释**：`.java` → `.class` 字节码 → JVM 解释或 JIT 为本地码
- 并非纯解释，热点代码会编译优化

## 面试一句话

> Java 面向对象、跨平台（JVM）、自动 GC、强类型与多线程，字节码一次编写到处运行。
